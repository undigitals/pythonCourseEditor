There are 6 main Operator types in Python which perform operations on variables and values.

# Arithmetic Operators

Arithmetic operators perform mathematical operations on numeric values.

Basic 4 arithmetic operators `Addition`, `Subtraction`, `Division`, `Multiplication` are used as same as they are used in Math.

```Python
x=7
y=15

print(x+y)
print(x*y)
print(y/x)
print(y-x)
```
